# Configuring Sounds

All the sounds that you intend to use will be stored on the server in there instance form. On startup this folder is read and converted into a table with all the information need to reconstruct any sound on the Server or Client.

## Setting SoundGroups

SoundGroups Are Automatically determined in the Sounds Folder, There is no need to manually set it yourself.

The First Set of children in the Sound folder must be Configs/Folders as the name of that folder is used to determine the sound group. And Sounds under the sub-folder is automatically grouped.
